🆘 HAVING TROUBLE?

   Issue: Scripts don't appear in ReaScript list
   Solution: Extensions → Refresh all scripts → Restart Reaper

   Issue: ReaPack download error
   Solution: Check internet → Try again → Repo might be temporarily down

   Issue: Script won't run
   Solution: Extensions → Show ReaScript console → Check error message
            Report on GitHub: https://github.com/OAlquimista33/reaper-scripts-collection/issues

   Still stuck?
   → Read INSTALACAO_COMPLETA.md (has detailed troubleshooting section)
